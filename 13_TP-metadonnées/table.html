Les métadonnées sont les données liées à un document qui ne sont pas affichées.

Méta-données
Balises Description
<meta name="keywords" content="tartiflette, saucisson, maroilles, …" /> elle permet d'associer des mots-clés à une page (OBSOLETE)
<meta name="generator" content="Nom de l’outil (version)" /> elle permet de faire des statistiques pour les éditeurs de code et logiciels de gestion de contenu (CMS)
<meta name="verify-v1" content="chaine aléatoire" /> elle permet de vérifier qu’un utilisateur est bien le propriétaire d’un site
<meta name="DC.Title" lang="fr" content="Un document en HTML" /> ignorées par les moteurs de recherches majeurs, elle peut être utile pour de l’archivage et de la gestion documentaire
<meta name="color-scheme" content="dark light"> définit les modes de couleur avec lesquels le document est compatible
<meta name="referrer" content="origin" seule l'origine du document est envoyée
